# Gameplay Notes / Idea Dump
When the thread is cut, the player loses
If the needle is hit on the side, the player loses

Losing can either be a "respawn at last stick" kind of thing or a complete level restart, really depends on how long I want to make the levels

When the needle sticks, it should start drawing thread from the wall (so the walls should be like stiched pillows and fabric stuff) (oooooh I can change the color of the thread to match!!!)
So if the thread is cut on an old line, it doesn't count as a loss state

But because it's drawing from a basically infinite source, the player shouldn't "run out" of linea

# Visuals & Polish
Needle should bend back and forth a little when it sticks (bend more if the hit was more powerful)

Needle should bend in direction the player is pulling back to launch (so it looks like it's storing force)
